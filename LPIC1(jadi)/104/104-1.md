## ğŸ“ Partitioning Disks with `fdisk` (MBR systems)
### ğŸ§¾ View partition table
```bash

```

### ğŸ› ï¸ Interactive Mode
```bash

```
- `m`: help
- `n`: new partition
- `d`: delete partition
- `t`: change partition type
- `p`: print partition table
- `w`: write changes
- `q`: quit without saving

### Partition Type IDs
| Code | Type                        |
|------|-----------------------------|
| 83   | Linux                      |
| 82   | Linux Swap / Solaris       |
| 5    | Extended Partition (MBR)   |
| EF   | EFI System (UEFI systems)  |

> ğŸ’¡ You can toggle bootable flag with `a`
